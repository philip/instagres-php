# Instagres PHP SDK documentation

These pages expand on the [root README](../README.md). Start there for install and a minimal example. You need **no Neon account or auth** to **create** a database or to **connect** with the URLs in the response. You **do** sign in to Neon only when you **claim** a database into your account. Unclaimed databases expire after **72 hours** unless you claim them for permanent use.

- [Migrate from 0.1.x](migration.md). Signature and return value changes in 0.2.0.
- [Usage guide](guide.md). Options when you create a database, lookups, connection URLs, claiming, and errors.
- [API reference](reference.md). Methods, return keys, and throws.
